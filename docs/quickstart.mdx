# Quickstart

Get up and running with **speakeasy** in under 5 minutes.

## Prerequisites

List any requirements:

- Node.js >= 18
- pnpm (recommended)

## Installation

```bash
# Install via pnpm
pnpm add speakeasy

# Or npm
npm install speakeasy
```

## Basic Usage

```typescript
// Example code
import { something } from 'speakeasy'

// Use it
something()
```
